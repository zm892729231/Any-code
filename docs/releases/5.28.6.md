## 更新内容

### 交互体验
- Plan 模式状态现在会按会话 / 项目维度持久化，减少多窗口切换后状态丢失的问题。
- 当你上滑查看历史消息时，流式输出不再强制把视图拉回底部，并新增“处理中 / 回到最新消息”提示。
- 优化消息列表自动滚动逻辑，历史会话首次加载时更稳定地定位到最新消息。

### 上下文与会话管理
- 自动压缩新增 `CompactionPending` 阶段，避免同一会话被重复触发压缩。
- 自动压缩改为复用原会话执行 `/compact`，减少伪会话和“没有历史可压缩”的异常情况。
- 会话列表会过滤 `/compact`、`/help`、`/cost` 等纯斜杠命令，以及内部自动压缩提示词，列表更干净。

### Git 与回滚
- 新增“禁用 Prompt 自动提交”设置，可保留工作区变更而不自动生成 Git commit。
- 改进 Rewind 能力判断，避免无实际代码变更时仍被错误标记为可回退。

### 稳定性
- 补充多处会话、Provider、WSL、MCP 相关稳定性修复与日志改进。
